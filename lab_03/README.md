# Лабораторна робота 3: Візуалізація даних

## Опис завдання
У цій лабораторній роботі реалізовано:
- Автоматизоване завантаження медичного датасету (Horse Colic) з репозиторію UCI.
- Очищення даних: виправлення структурного зсуву колонок та парсинг сирих даних.
- Обробку пропущених значень шляхом імп'ютації: медіаною для числових атрибутів та модою для категоріальних.
- Оптимізацію та приведення типів даних до відповідних форматів (`int`, `float`).
- Побудову багатовимірних візуалізацій за допомогою `matplotlib` та `seaborn`:
  - **Boxplot:** Аналіз розподілу температури залежно від рівня болю.
  - **Histplot:** Гістограма розподілу пульсу за 5 заданими діапазонами.
  - **Scatter 3D:** Тривимірна діаграма розсіювання тріади життєвих показників (пульс, частота дихання, температура).
  - **Heatmap:** Теплова карта кореляції для всіх числових атрибутів.
  - **Pie Chart:** Кругова діаграма загальної статистики результатів лікування.

## Вимоги до системи
- ОС: Windows, macOS або Linux
- Інтерпретатор: Python 3.9 або вище (рекомендовано 3.10+)
- Встановлений Jupyter Notebook або інше середовище для запуску `.ipynb` файлів.

## Інструкція із встановлення та запуску

1. Склонуйте репозиторій на свій комп'ютер та перейдіть у директорію проєкту:
   ```bash
   git clone https://github.com/alyaalonachwenous-stack/labs_da.git
   cd labs_da/lab_03
   ```

2. Створіть та активуйте віртуальне середовище:
   ```bash
    python -m venv venv
   ```
    # Для Windows:
   ```bash
    venv\Scripts\activate
   ```
    # Для macOS/Linux:
   ```bash
    source venv/bin/activate
   ```

3. Встановіть усі необхідні бібліотеки із файлу залежностей однією командою:
   ```bash
   pip install -r requirements.txt
   ```

4. Запустіть Jupyter Notebook для перегляду та виконання коду (завантаження необхідних датасетів реалізовано автоматично всередині скриптів):
   ```bash
   jupyter notebook
   ```
