# Лабораторна робота 6: Обробка відео з камери за допомогою OpenCV мовою C++

## Опис завдання
У цій лабораторній роботі реалізовано:
- Захоплення та відображення відеопотоку з веб-камери в реальному часі.
- Сувору модульну об'єктно-орієнтовану архітектуру: логіка розділена на незалежні класи (`CameraProvider`, `KeyProcessor`, `FrameProcessor`, `Display`) з окремими `.hpp` та `.cpp` файлами.
- Чотири режими обробки зображень (перемикання "на льоту"):
  - **Оригінальний потік**
  - **Інверсія кольорів** (`bitwise_not`)
  - **Gaussian Blur** (розмиття)
  - **Canny фільтр** (виділення контурів)
- Інтерактивні елементи керування:
  - **Повзунок (Trackbar):** для динамічного налаштування інтенсивності розмиття (для Blur) або порогів чутливості (для Canny).
  - **Миша:** малювання червоним маркером по відео (затискання лівої кнопки) та миттєве очищення екрана від малюнків (клік правою кнопкою).
- Відображення телеметрії: вивід поточного активного режиму та лічильника FPS (кадрів на секунду) прямо на відеокадрі.
- Надійну систему завершення роботи: програма коректно закривається як через гарячі клавіші, так і по кліку на системний "хрестик" вікна.
- Автоматизовану систему збірки через `CMake` та зручні Bash-скрипти для розгортання проєкту на чистій Linux-системі з першої спроби.

## Вимоги до системи
- **ОС:** Linux (рекомендовано Ubuntu/Debian)
- **Залізо:** Наявність підключеної та налаштованої веб-камери (для віртуальних машин потрібен USB passthrough).
- **ПЗ:** C++17, CMake, Make, GCC.

## Інструкція із встановлення та запуску

1. Склонуйте репозиторій на свій комп'ютер та перейдіть у директорію проєкту:
   ```bash
   git clone https://github.com/alyaalonachwenous-stack/labs_da.git
   cd labs_da/lab_06
   ```

2. Надайте права на виконання всім Bash-скриптам:
   ```bash
   chmod +x *.sh
   ```

3. Встановіть необхідні залежності та бібліотеки (OpenCV, CMake, GCC). *Потребує прав sudo*:
   ```bash
   ./preinstall.sh
   ```

4. Зберіть проєкт. Скрипт автоматично створить директорію `build`, згенерує файли CMake у режимі Release та скомпілює програму:
   ```bash
   ./build.sh
   ```

5. Запустіть готову програму:
   ```bash
   ./run.sh
   ```

## Керування програмою
- **`1`** — Звичайний режим (без фільтрів)
- **`2`** — Режим інверсії кольорів
- **`3`** — Режим Gaussian Blur
- **`4`** — Режим Canny Edge
- **`ЛКМ` (затиснути)** — Малювання червоних ліній по відео
- **`ПКМ` (клік)** — Очищення всіх намальованих ліній
- **Повзунок** — Зміна параметрів для режимів 3 та 4
- **`ESC` / `Q` / Хрестик вікна** — Завершення роботи програми
