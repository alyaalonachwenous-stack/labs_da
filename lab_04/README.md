# Лабораторна робота 4: Візуалізація даних 2

## Опис завдання
У цій лабораторній роботі реалізовано:
- Генерацію чистого гармонічного сигналу за математичною функцією $y(t) = A \cdot \sin(\omega t + \phi)$.
- Програмне накладання нормального (гауссівського) шуму з керованими параметрами середнього значення та дисперсії. Форма шуму зафіксована (`seed`), що запобігає його хаотичній регенерації при зміні параметрів гармоніки.
- Застосування низькочастотного фільтра Баттерворта (`scipy.signal.butter`) для ефективного очищення зашумленого сигналу.
- Побудову інтерактивного графічного інтерфейсу (GUI) за допомогою модуля `matplotlib.widgets`, який включає:
  - **Sliders:** 6 повзунків для керування параметрами в реальному часі (амплітуда, частота, фаза, середнє шуму, дисперсія шуму, частота зрізу фільтра).
  - **CheckButtons:** Чекбокс для миттєвого увімкнення/вимкнення відображення шуму на графіку.
  - **Button:** Кнопка «Reset» для скидання всіх повзунків до їх початкових (дефолтних) значень через систему подій (callbacks).
- Одночасну візуалізацію трьох станів сигналу на одному графіку (чиста гармоніка, зашумлений сигнал, відфільтрований сигнал) для зручного порівняльного аналізу.

## Інструкція із встановлення та запуску

1. Склонуйте репозиторій на свій комп'ютер та перейдіть у директорію проєкту:
   ```bash
   git clone https://github.com/alyaalonachwenous-stack/labs_da.git
   cd labs_da/lab_03
   ```

2. Створіть та активуйте віртуальне середовище:
   ```bash
    python -m venv venv
   ```
    # Для Windows:
   ```bash
    venv\Scripts\activate
   ```
    # Для macOS/Linux:
   ```bash
    source venv/bin/activate
   ```

3. Встановіть усі необхідні бібліотеки із файлу залежностей однією командою:
   ```bash
   pip install -r requirements.txt
   ```

4. Запустіть скрипт для відкриття інтерактивного вікна програми
