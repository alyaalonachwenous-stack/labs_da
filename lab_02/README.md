# Лабораторна робота 2: Аналіз та перетворення даних

## Опис завдання
У цій лабораторній роботі реалізовано:
- Фільтрацію даних за допомогою бібліотеки Pandas.
- Очищення даних від артефактів (знаків питання) та видалення пропущених значень (NaN).
- Нормалізацію та стандартизацію числових атрибутів за допомогою `scikit-learn`.
- Розрахунок кореляції Пірсона та Спірмена.
- Застосування One-Hot Encoding (OHE) для категоріальних даних.

## Вимоги до системи
- ОС: Windows, macOS або Linux
- Інтерпретатор: Python 3.9 або вище (рекомендовано 3.10+)
- Встановлений Jupyter Notebook або інше середовище для запуску `.ipynb` файлів.
