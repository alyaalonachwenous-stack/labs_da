# Лабораторна робота 5: Наука про дані: обмін результатами та початковий аналіз


## Опис завдання
У цій лабораторній роботі реалізовано:
- Аналіз історичних даних NOAA (індекси здоров'я рослинності VCI, TCI, VHI) для різних областей України.
- Побудову інтерактивного веб-додатка за допомогою фреймворку Streamlit, який включає зручну панель керування (Sidebar) та головний екран відображення результатів.
- Інтерактивні елементи керування:
   - Dropdown списки: для вибору досліджуваного індексу та області.
   - Sliders (повзунки): для налаштування часових інтервалів (фільтрація за діапазоном років та тижнів).
   - Checkboxes (чекбокси): для сортування даних у таблиці за зростанням або спаданням. Реалізовано професійну логіку взаємовиключення через st.session_state (при увімкненні одного чекбокса інший автоматично вимикається).
   - Button (кнопка): «Скинути всі фільтри» для миттєвого повернення інтерфейсу до дефолтних значень за допомогою системи Callbacks (on_click).
- Організацію відображення результатів через систему вкладок (st.tabs):
 - Таблиця: перегляд відфільтрованих та відсортованих даних.
 - Графік динаміки: візуалізація часового ряду для обраної області з автоматичним пошуком та підсвічуванням аномалій (історичний мінімум та максимум), а також опціональним відображенням середньої норми по області.
 - Графік порівняння: комплексна візуалізація, що показує динаміку обраної області на фоні всіх інших областей України. Включає додатковий випадаючий список для вибору "області-конкурента" (з опцією "Немає") для прямого порівняння графіків.

## Інструкція із встановлення та запуску

1. Склонуйте репозиторій на свій комп'ютер та перейдіть у директорію проєкту:
   ```bash
   git clone https://github.com/alyaalonachwenous-stack/labs_da.git
   cd labs_da/lab_03
   ```

2. Створіть та активуйте віртуальне середовище:
   ```bash
    python -m venv venv
   ```
    # Для Windows:
   ```bash
    venv\Scripts\activate
   ```
    # Для macOS/Linux:
   ```bash
    source venv/bin/activate
   ```

3. Встановіть усі необхідні бібліотеки із файлу залежностей однією командою:
   ```bash
   pip install -r requirements.txt
   ```

4. Запустіть локальний сервер Streamlit у терміналі для відкриття інтерактивного веб-додатка у вашому браузері:
   ```bash
   streamlit run lab_05.py
   ```
